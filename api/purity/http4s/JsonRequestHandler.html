    Runs a Script function (A => Script[D, E, B]) to handle an http request and produce a response for http4s.

    Runs a Script function (A => Script[D, E, B]) to handle an http request and produce a response for http4s. This function also handle format of responses and handles lower level errors (failed F monads). The configuration and logger required by the Script should be implicitly provided, as well as a function that should map the Script failures to valid http4s responses.
