HttpClient 

case class HttpClient(client: Client[IO]) extends Product with Serializable

Intended to be used inside services which are dependencies. - Lifts http4s client operations into the Script type. - Adds logging between requests and responses. - Adds possible IO failures: ServiceUnexpectedStatus, ServiceUnexpectedEntity and ServiceUnexpectedJson
