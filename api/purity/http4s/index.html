    Intended to be used inside services which are dependencies.

    Intended to be used inside services which are dependencies. Basic configuration of a service. Can create the correct org.http4s.Uri of the endpoints.

    Note: This has impure methods that throw exceptions when trying to build the URLs and URIs, this is because this objects should be instantiated when booting the server, and the server should NOT boot if the configured endpoints are invalid.

    Exceptions to be used to fail IO monads when requesting other services. They add more data about the failure that can be used for logging or creating more sensible responses to the client.
